🩺 DocConnect
A MERN Stack Doctor Appointment Booking System
DocConnect is a full-stack healthcare web application built using the MERN stack.
It enables patients to book appointments seamlessly while allowing doctors and administrators to efficiently manage schedules.

📌 Problem Statement
Traditional appointment booking systems are manual and time-consuming.
DocConnect digitizes the process, improving accessibility, efficiency, and user experience.

✨ Key Features
👤 Patient
Secure Authentication (JWT-based)
Book & Cancel Appointments
View Appointment History
👨‍⚕️ Doctor
Manage Availability
Approve / Reject Appointments
View Patient Details
🛠 Admin
Manage Doctors & Users
Monitor System Activity
🏗 Architecture
Frontend → React.js
Backend → Node.js + Express.js
Database → MongoDB
State Management → Context API / Redux (if used)
